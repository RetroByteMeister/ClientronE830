## Usage/Examples

Simply download the files you need. This archive includes the following directories:

### DOS622/

All MS-DOS v6.22 tools that I showed in the video. Will likely work on any DOS-compatible computer.

### WinXP/

Drivers specific for the Clientron E830 for use with Windows XP. This directory also includes a virus scanner that as of 2025 still receives definition updates as well as an installer of [Legacy Update](https://legacyupdate.net/) so you can install latest security updates on Windows. 
**Warning:** It is not recommended to go online with legacy hardware and outdated operating systems like DOS or windows XP. This archive is for offline retro gaming only. 

### img/

Premade, bootable DOS images that you can write to a Compact Flash card or USB thumb drive with a tool like [Balena Etcher](https://etcher.balena.io/). Has everyhing ready to go, including sound support.
